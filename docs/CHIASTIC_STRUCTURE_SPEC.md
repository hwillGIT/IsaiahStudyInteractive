# Chiastic Structure Modal Specification

## Overview
This specification defines the design pattern for implementing chiastic (symmetrical) structure displays in the Isaiah study application. Chiastic structures (A-B-C-B'-A') emphasize key themes through literary symmetry, with parallel sections commenting on and enriching each other.

## Design Pattern for Structure Modals

### Visual Styling Guidelines

1. **Consistent Modal Format**
   - Max width: `max-w-3xl`
   - Max height: `max-h-[85vh]`
   - Compact mono-spaced font display: `font-mono text-xs`
   - Background: `bg-gray-50` for structure block
   - Padding: `p-4` for structure block

2. **Color-Coded Sections**
   - Use small colored dots: `w-3 h-3 rounded mt-0.5 flex-shrink-0`
   - Colors must match verse grid colors exactly
   - Bold, colored titles using `font-sans font-semibold` with matching color classes

3. **Indentation Pattern for Symmetrical Chapters**
   - **Top half** (approaching center): Indent inward (ml-0 → ml-4 → ml-8)
   - **Center**: Highlighted background (yellow, blue, or green tint) with border-l-4
   - **Bottom half** (moving from center): Indent outward (ml-8 → ml-4 → ml-0)
   - **Border separator**: `mt-3 border-t-2 border-gray-300 pt-2` between top and bottom

4. **Indentation Pattern for Linear Chapters**
   - Consistent horizontal alignment
   - Hinge/turning point: Highlighted background + border-l-4
   - Subsequent sections: Same indentation level showing forward flow

5. **Center/Hinge Highlighting**
   - Background: `bg-yellow-100`, `bg-blue-100`, or `bg-green-100` (theme-appropriate)
   - Border: `border-l-4 border-[color]-400` or `border-[color]-600`
   - Padding: `px-2 py-1 rounded`
   - Text: `font-sans font-bold text-[color]-800`
   - Symbol: `★` prefix for visual emphasis

### Theological Connection Display

#### For Symmetrical Chapters (A-B-C-B'-A')

**Pattern:**
```
Structure Diagram (shows indentation pattern)
↓
Explanatory text about parallel sections
↓
"Theological Connections:" section with bullet points
```

**Connection Format:**
- **A ↔ A'**: [Description of how opening and closing mirror each other]
- **B ↔ B'**: [Description of thematic connection]
- **C (Center)**: [Why this is the theological pivot]

**Example Language** (devotional, not academic):
- "The opening vision of glory (A) finds its answer in the closing judgment (A')"
- "God's call to repentance (B) is matched by His promise of restoration (B')"
- "At the center stands the transforming moment when..."

#### For Linear Chapters (with Hinge)

**Pattern:**
```
Structure Diagram (shows linear flow)
↓
Explanatory text about how the hinge transforms the narrative
↓
"How This Unfolds:" section describing the progression
```

**Progression Format:**
- **Before the turning point**: [Setup/problem]
- **Turning point**: [What changes]
- **After the turning point**: [Consequences/response]

### Content Guidelines

1. **Language Style**
   - Use everyday language, not academic jargon
   - Avoid terms like "chiasm," "pericope," "literary device"
   - Use instead: "symmetrical pattern," "mirror structure," "turning point," "center"

2. **Theological Focus**
   - Show how parallel sections interpret each other
   - Emphasize God's character and actions
   - Connect to the gospel/Christ where appropriate
   - Make it devotionally rich, not just structurally informative

3. **Verse References**
   - Use parenthetical verse references: (2:1-5), (6:8), etc.
   - Include short quoted phrases in italics
   - Example: *"Holy, holy, holy is the LORD of hosts"*

4. **Symmetry Visualization**
   - Use arrows: ↔ for parallel connections
   - Use indentation to show literary flow
   - Keep visual elements simple and clean

## Consistency Standards

1. All chapters use same modal dimensions and styling
2. All colored dots are exactly `w-3 h-3 rounded mt-0.5 flex-shrink-0`
3. All section titles use `font-sans font-semibold text-[color]-700`
4. All center highlights use background tint + border-l-4 + bold text
5. All explanatory text uses everyday language
6. All theological connections are clearly stated with ↔ arrows for parallels

## Avoid These Mistakes

❌ Using academic terms: "chiasm," "pericope," "literary structure"
❌ Colored boxes/panels (use small dots instead)
❌ Inconsistent indentation patterns
❌ Missing theological explanations
❌ Colors that don't match verse grid
❌ Overly complex diagrams
❌ Jargon-heavy explanations

✅ Use simple language: "symmetrical pattern," "turning point," "mirror structure"
✅ Use small colored dots consistently
✅ Follow indentation patterns for symmetry
✅ Always explain the theological significance
✅ Match colors to verse grid exactly
✅ Keep diagrams clean and minimal
